# Regras

Esta pasta contém as **Regras Éticas e Estratégicas** que governam o projeto BolSin.

Inclui diretrizes sobre:

*   **Ética:** Princípios de conduta e transparência.
*   **Estratégia:** Limites e fronteiras operacionais (o que faremos e o que *não* faremos).

Estas regras são essenciais para manter a consistência, a confiança e a direção estratégica do projeto.
