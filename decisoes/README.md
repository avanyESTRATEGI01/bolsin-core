# Decisões (Registro Estratégico)

Esta pasta é o **Registro de Decisões Estratégicas** do projeto.

**Instruções para a IA (Manus):**

1.  **Registro:** Embora as decisões possam ser iniciadas como Issues (como a Decisão #1), o registro final e formal deve ser armazenado aqui em arquivos Markdown.
2.  **Formato:** Cada decisão deve ser salva em um arquivo Markdown separado (ex: `decisao-001-foco-instagram.md`) contendo: Título, Motivo, Impacto, Data e Status.
3.  **Consulta:** Use esta pasta como a fonte de verdade para entender o histórico e a evolução da estratégia do BolSin. Antes de propor uma nova ação, verifique se ela não contradiz uma decisão estratégica já registrada aqui.

**Nota:** A Issue #1 (Foco em Instagram) deve ser formalizada em um arquivo Markdown nesta pasta.
