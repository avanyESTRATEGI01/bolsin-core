# Dados (Base de Dados Versionada)

Esta pasta é o repositório da **Base de Dados Versionada** do projeto.

**Instruções para a IA (Manus):**

1.  **Armazenamento:** Use esta pasta para armazenar dados brutos ou processados (ex: CSV, JSON, planilhas) que precisam ser versionados junto com o código e a estratégia.
2.  **Análise:** Ao realizar análises, utilize os arquivos desta pasta como fonte primária de dados.
3.  **Atualização:** **NUNCA** altere arquivos de dados diretamente. Se for necessário adicionar novos dados ou atualizar um conjunto, crie um novo arquivo com um timestamp ou número de versão claro e documente a alteração em um commit detalhado. Isso garante a rastreabilidade e a integridade dos dados.
