# Funnels (Jornada de Conteúdo e Conversão)

Esta pasta armazena a organização da **Jornada de Conteúdo e Conversão** do BolSin.

**Instruções para a IA (Manus):**

1.  **Consulta:** Use esta pasta para entender a estrutura atual dos funis (ex: Topo, Meio, Fundo de Funil) e a progressão do usuário.
2.  **Criação:** Ao gerar conteúdo ou estratégias, garanta que o material esteja alinhado com a etapa do funil a que se destina.
3.  **Atualização:** Se for necessário propor uma nova estrutura de funil ou otimizar uma existente, documente a proposta em um novo arquivo Markdown aqui e crie uma Issue para aprovação.
