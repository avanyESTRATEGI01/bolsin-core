# Pilares

Esta pasta armazena os **Princípios Inegociáveis** que guiam o produto, a comunicação e a operação do BolSin.

Os pilares servem como um **filtro de decisão** para garantir que todas as ações e desenvolvimentos estejam alinhados com os valores centrais do projeto. Qualquer nova funcionalidade ou campanha de comunicação deve ser avaliada à luz destes pilares.
