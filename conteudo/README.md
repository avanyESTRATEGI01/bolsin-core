# Conteúdo (Ideias e Roteiros)

Esta pasta é o *hub* criativo para **Ideias e Roteiros de Conteúdo**.

**Instruções para a IA (Manus):**

1.  **Geração:** Ao gerar novas ideias de conteúdo (posts, vídeos, artigos), armazene-as aqui em arquivos Markdown ou de texto simples, seguindo uma estrutura de nomeação clara (ex: `ideia-instagram-001.md`).
2.  **Organização:** Mantenha os roteiros e rascunhos organizados em subpastas se necessário (ex: `conteudo/instagram`, `conteudo/blog`).
3.  **Referência:** Antes de criar um novo conteúdo, consulte esta pasta para evitar duplicação e garantir que o novo material se baseie em ideias já aprovadas ou em desenvolvimento.
